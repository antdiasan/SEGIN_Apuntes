# Filtrado mediante proxys

El **filtrado mediante proxys** es una técnica que permite controlar y gestionar el acceso a Internet desde una red, actuando como intermediario entre los usuarios y los servicios externos.  
Este tipo de filtrado se utiliza para **mejorar la privacidad, la seguridad y el control del tráfico** de red.

El uso de proxys es habitual en entornos educativos, empresariales y redes gestionadas.

El proxy permite reducir tráfico repetido mediante caché.

---

## Qué es un proxy

Un proxy es un sistema que recibe las peticiones de los usuarios y las reenvía a Internet en su nombre.

De esta forma, los dispositivos de la red no se conectan directamente a los servicios externos, sino a través del proxy, que actúa como punto de control.

Este funcionamiento permite aplicar políticas de filtrado y supervisión del tráfico.

---

## Filtrado de contenidos mediante proxys

El proxy puede utilizarse para **permitir o bloquear** el acceso a determinados contenidos o servicios.

Este filtrado se basa en reglas que pueden tener en cuenta aspectos como:

- El tipo de contenido  
- El sitio web de destino  
- El horario de acceso  

El objetivo no es espiar al usuario, sino **gestionar el uso de la red** de forma responsable.

---

!!! info "Importante"
    El filtrado mediante proxys debe aplicarse de forma transparente y respetando la normativa de protección de datos.

---

## Proxys y privacidad

El uso de un proxy puede mejorar la privacidad al ocultar la dirección IP real del usuario frente a los servicios externos.

Sin embargo, el propio proxy tiene acceso al tráfico que gestiona, por lo que debe ser un sistema confiable y correctamente administrado.

La privacidad depende tanto de la tecnología como de la gestión que se haga de ella.


!!! warning "Peligro"
    Un proxy mal configurado o de origen desconocido puede comprometer la **privacidad y la seguridad** de las comunicaciones.


!!! abstract "Idea clave"
    Un proxy puede mejorar la privacidad frente a Internet, pero concentra la confianza en un único punto de la red.


---

## Uso de proxys en redes gestionadas

En redes educativas y empresariales, los proxys se utilizan para:

- Controlar el acceso a Internet  
- Reducir el consumo de ancho de banda  
- Mejorar la seguridad frente a contenidos no deseados  

Este tipo de filtrado ayuda a mantener un entorno de red más seguro y controlado.


!!! info "Herramientas profesionales"
    En entornos Linux, el software más utilizado para implementar un servidor proxy con filtrado de contenidos es **Squid**. Permite definir listas negras de dominios y filtrar contenidos de forma centralizada para toda una red educativa o empresarial.
